# Alef-Null

Математический мод для Mindustry. Добавляет числа как ресурсы. Первый шаг к математической вселенной.

## Что добавляет сейчас (v0.02)

- Ресурс: Единица (1) - зелёная монетка (вообще ничего не делает)

## Важно

Мод в активной разработке. Контента пока немного (только монетка), но он будет расширяться с каждой версией.

## Установка

1. Скачай `Alef-Null.zip` из [раздела Releases](https://github.com/oscillatorium/Alef-Null/releases) на GitHub
2. В игре: Моды -> Импорт мода -> Локально
3. Выбери скачанный ZIP-архив (не распаковывай)

Мод скоро появится в Браузере модификаций прямо в игре (может быть).
